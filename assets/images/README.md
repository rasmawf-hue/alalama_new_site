# مجلد الصور

## التعليمات:

عند تثبيت القالب على WordPress:

1. قم بتحميل الصور من [Google Drive](https://drive.google.com/drive/folders/1HV3qRSZtOtcAV4hQ9i511C_iiRlT3oFJ)
2. ارفعها إلى **الوسائط** في WordPress
3. استخدمها في الصفحات

## البديل:

يمكنك رفع الصور هنا في GitHub بالهيكل التالي:

images/
├── logo/
│ └── logo.png
├── hero/
├── services/
├── portfolio/
└── clients/

## الصور المطلوبة:

- شعار الشركة (logo)
- صور الخدمات
- صور معرض الأعمال
- شعارات العملاء
